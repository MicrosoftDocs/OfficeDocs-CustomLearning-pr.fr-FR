# <a name="translate-learning-pathways-custom-playlists"></a>Traduire les sélections personnalisées des parcours d’apprentissage
Avec le parcours d'apprentissage, la façon dont vous travaillez avec des sélections personnalisées dépend de la façon dont vous définissez les parcours d'apprentissage.  

- Si vous avez mis en service un nouveau site multilingue de parcours d'apprentissage, vous pouvez copier ou créer une nouvelle playlist et la traduire dans les langues que le site prend en charge.
- Si vous avez mis à niveau un site de parcours d'apprentissage existant qui contient des sélections personnalisées pré-existantes, vous pouvez traduire les sélections existantes dans les langues que votre site prend en charge. Vous pouvez également copier et créer des playlists. 

> [!IMPORTANT]
> - Les playlists fournies par Microsoft sont déjà traduites et ne peuvent pas être modifiées, sauf si vous créez une copie de la playlist. 

## <a name="create-or-copy-a-playlist-and-have-it-translated"></a>Créer ou copier une playlist et la traduire
Si vous avez mis en service un nouveau site multilingue de parcours d'apprentissage ou mis à niveau le site avec le site web multilingue, vous pouvez copier ou créer une playlist personnalisée et traduire les ressources de playlist et de playlist dans les langues pris en charge par votre site. 

### <a name="copy-a-playlist-and-have-it-translated"></a>Copier une playlist et la traduire
Pour montrer comment créer une playlist personnalisée et la traduire, nous allons utiliser la fonctionnalité Copier la playlist du parcours d'apprentissage.

1.  Dans le **menu** Accueil du parcours d'apprentissage, cliquez sur Administration du parcours **d'apprentissage.** 
2.  Sous **microsoft 365 parcours** d'apprentissage, sélectionnez Premiers jours, puis sélectionnez Six étapes simples - Bienvenue dans Microsoft **365**.  
3. Cliquez **sur Copier la sélection,** faites défiler la page vers le bas et cliquez sur Modifier les **détails.**   
4. Modifiez le titre de la playlist. Dans cet exemple, nous allons modifier le titre de la playlist en **cinq étapes simples.** Les langues disponibles dépendent des langues prise en charge sélectionnées pour le site. Pour ajouter des langues supplémentaires, vous devez ajouter d'autres langues prise en charge au site.   
5. Cliquez **sur Enregistrer les détails.** 
6. Cliquez sur une langue à droite de la langue par défaut (dans ce cas, l'anglais) pour afficher les détails de la sélection copiée. Les détails de la nouvelle langue reflètent la playlist copiée d'origine et nécessitent une traduction.
7. Prenez note pour informer le traducteur des détails de la playlist à traduire.  

![Exemple de page italien.](media/custom_update_ml_pldetail.png)

> [!IMPORTANT]
> Les notifications du traducteur ne sont pas intégrées dans des playlists personnalisées. Les traducteurs doivent être avertis manuellement. Les traducteurs traduisent manuellement les copies des détails de langue par défaut dans la ou les langues spécifiées. Vous devez informer le traducteur des traductions requises pour les détails de la playlist. Il est recommandé de terminer toutes les modifications de playlist, y compris l'ajout, la modification ou la suppression de ressources de sélection, puis d'informer le traducteur des traductions requises pour les détails et les ressources de la sélection.

## <a name="create-multilingual-assets-in-a-custom-playlist"></a>Créer des ressources multilingues dans une sélection personnalisée
Les ressources d'une playlist fournies par Microsoft ne sont pas modifiables et ne nécessitent pas de traduction. Les ressources fournies par Microsoft seront disponibles dans les langues pris en charge par le site. Pour les ressources personnalisées que vous créez ou les ressources personnalisées existantes, vous pouvez ajouter les langues pris en charge par le site.  Examinons comment ajouter des langues pour les ressources personnalisées nouvelles et existantes dans une playlist. 

### <a name="create-a-new-multilingual-asset-for-a-playlist"></a>Créer une nouvelle fonctionnalité multilingue pour une playlist
Ces instructions supposent que vous utilisez la playlist Cinq étapes simples dans l'exemple de cette rubrique. 

**Pour créer le bien**
1. Pour la playlist Cinq **étapes** simples, sous Ressources, sélectionnez **+ Nouveau** bien, puis remplissez les détails du bien. 
   - Pour **titre,** entrez « Bienvenue dans Microsoft 365 chez Contoso ». 
   - Pour **la** technologie, **sélectionnez Microsoft 365**.
![Formulaire dans lequel vous entrez les détails des biens.](media/custom_ml_pl_createasset.png)
2. Sélectionnez **Créer un bien,** puis **sélectionnez Enregistrer le bien.**

**Pour ouvrir le bien**
1. Sélectionnez le bien que vous avez créé, puis cliquez sur **Ouvrir.**  
2. A apporté les modifications souhaitées à la page. Pour l'instant, vous pouvez modifier le titre de la page et ajouter du texte à la page, comme illustré dans l'exemple suivant, puis cliquer sur **Publier**. 
3. Cliquez **sur Traduction.**
4. Sélectionnez **Créer** pour la ou les langues de votre choix. Pour cet exemple, nous allons sélectionner l'italien.
5. Sélectionnez **Afficher**. Vous devez maintenant voir une page semblable à l'exemple suivant en fonction de vos langues pris en charge.
![Exemple de page Éléments de sélection.](media/custom_ml_pl_createasset.png)
6. Cliquez sur **Publier**.
7. Pour la page de traduction que vous avez créée, copiez l'URL à partir du navigateur et revenir à la page d'administration du parcours d'apprentissage.

**Pour ajouter une langue pour le bien de traduction que vous avez créé** 
1. Revenir à la page Administration du parcours d'apprentissage et sélectionnez l'icône Modifier (crayon) pour le bien « Cinq étapes simples » que vous avez ouvert. 
2. Cliquez **sur Ajouter** une langue, puis sélectionnez une langue. Dans cet exemple, nous avons sélectionné l'italien. 
3. Entrez l'URL de la page traduite que vous avez créée. Vous devriez voir une page semblable à l'exemple suivant. 
![Formulaire dans lequel vous entrez l'URL d'une page de langue spécifique](media/custom_ml_pl_translateasseturl.png)
4. Cliquez **sur Enregistrer le bien.**

## <a name="add-languages-for-existing-custom-playlists-and-assets"></a>Ajouter des langues pour les sélections et ressources personnalisées existantes
Si vous avez mis à niveau les parcours d'apprentissage pour la prise en charge multilingue et que vous avez des playlists et des ressources personnalisées existantes, vous pouvez ajouter des langues prise en charge pour les playlists et les ressources. Ces instructions supposent que vous avez une playlist personnalisée à laquelle vous pouvez ajouter des langues. Si vous n'avez pas de playlist personnalisée, voir « Créer ou copier une playlist » plus tôt dans cette rubrique. 

> [!IMPORTANT]
> - Les playlists fournies par Microsoft sont déjà traduites et ne peuvent pas être modifiées, sauf si vous créez une copie de la playlist. Vous pouvez uniquement ajouter des langues pour les playlists personnalisées. 
> - Pour ajouter des langues pour des playlists et des ressources personnalisées, la prise en charge multilingue et les langues doivent être activées. Pour plus d'informations, voir [Définir les paramètres de langue.](./custom_update_ml.md#set-language-settings) 

### <a name="to-add-languages-for-an-existing-playlist"></a>Pour ajouter des langues pour une playlist existante
1.  Dans le **menu** Accueil du parcours d'apprentissage, cliquez sur Administration du parcours **d'apprentissage.** 
2.  Sélectionnez une playlist personnalisée. Une playlist personnalisée est identifiée par l'icône personnes avec la coche, comme illustré dans l'exemple suivant.
![Exemple de playlist personnalisée.](media/custom_ml_pl_customselect.png) 
3. Sélectionnez **Modifier les détails.** Vous devrez peut-être faire défiler vers le bas pour trouver **le bouton Modifier les détails.**   
4. Sélectionnez **Ajouter une** langue, puis sélectionnez une langue.  
5. Sélectionnez **Enregistrer les détails.**
 Les langues disponibles dépendent des langues prise en charge sélectionnées pour le site. Pour ajouter des langues supplémentaires, vous devez ajouter d'autres langues prise en charge au site.   
5. Cliquez sur une langue à droite de la langue par défaut (dans ce cas, l'anglais) pour afficher les détails de la sélection copiée. Les détails de la nouvelle langue reflètent la playlist copiée d'origine et nécessitent une traduction.
![Exemple de page de détails pour l'italien.](media/custom_ml_pl_editdetailit.png) 
7. Prenez note pour informer le traducteur des détails de la playlist à traduire.  

**Pour ouvrir le bien**
1. Sélectionnez le bien que vous avez créé, puis cliquez sur **Ouvrir.**  
2. A apporté les modifications souhaitées à la page. Pour l'instant, vous pouvez modifier le titre de la page et ajouter du texte à la page, comme illustré dans l'exemple suivant, puis cliquer sur **Publier**. 
3. Cliquez **sur Traduction.**
4. Sélectionnez **Créer** pour la ou les langues de votre choix. Pour cet exemple, nous allons sélectionner l'italien.
5. Sélectionnez **Afficher**. Vous devez maintenant voir une page semblable à l'exemple suivant en fonction de vos langues pris en charge.
![Exemple de page qui s'affiche lorsque vous sélectionnez Afficher.](media/custom_ml_pl_createasset.png)
6. Cliquez sur **Publier**.
7. Pour la page de traduction que vous avez créée, copiez l'URL à partir du navigateur et revenir à la page d'administration du parcours d'apprentissage.

## <a name="notify-the-translator"></a>Avertir le traducteur
Lorsque vous avez terminé de créer des ressources de traduction, informez le traducteur des traductions requises. Le traducteur :
- Traduire les détails de la playlist.
- Traduire les détails des biens.
- Traduire les pages de langue ajoutées pour un bien.
- Informer le demandeur des traductions que les traductions sont prêtes pour révision

## <a name="add-languages-for-an-existing-custom-playlist"></a>Ajouter des langues pour une playlist personnalisée existante
Si vous avez mis à niveau les parcours d'apprentissage pour la prise en charge multilingue et que vous avez des playlists et des ressources personnalisées existantes, vous pouvez ajouter des langues prise en charge pour les playlists et les ressources. 

> [!IMPORTANT]
> - Les playlists fournies par Microsoft sont déjà traduites et ne peuvent pas être modifiées, sauf si vous créez une copie de la playlist. Vous pouvez uniquement ajouter des langues pour les playlists personnalisées. 
> - Pour ajouter des langues pour des playlists et des ressources personnalisées, la prise en charge multilingue et les langues doivent être activées. Pour plus d'informations sur l'activation de la prise en charge et des options multilingues, voir [Définir les paramètres de langue.](./custom_update_ml.md#set-language-settings) 


## <a name="instructions-for-the-translator"></a>Instructions pour le traducteur
Lorsque vous ajoutez des langues aux playlists, vous devez informer le traducteur des modifications. Vous pouvez faire pointer le traducteur vers les points suivants : 

### <a name="translate-playlist-details"></a>Traduire les détails de la playlist
Dans le **menu** Accueil du parcours d'apprentissage, cliquez sur Administration du parcours **d'apprentissage.** 
1. Cliquez sur la playlist personnalisée qui nécessite une traduction, puis cliquez sur les langues. 
2. Cliquez **sur Modifier les détails,** effectuer les traductions de la playlist, puis cliquez sur 
3. Cliquez **sur Enregistrer les détails.** 
4. Informez le demandeur de traduction que la traduction est terminée. 

### <a name="translate-asset-details"></a>Traduire les détails des biens
Dans le **menu** Accueil du parcours d'apprentissage, cliquez sur Administration du parcours **d'apprentissage.** 
1. Cliquez sur la playlist personnalisée qui nécessite une traduction. 
2. Faites défiler la page vers le bas, puis sous Ressources, sélectionnez Modifier la valeur de l'actif à modifier, puis sélectionnez la langue. 
3. Faites les traductions de la bien, puis cliquez sur **Enregistrer le bien.**  

### <a name="translate-the-added-language-page-for-the-asset"></a>Traduire la page de langue ajoutée pour le bien
Dans le **menu** Accueil du parcours d'apprentissage, cliquez sur Administration du parcours **d'apprentissage.** 
1. Cliquez sur la playlist personnalisée qui nécessite une traduction. 
2. Faites défiler la page vers le bas, puis sous Ressources, sélectionnez l'actif, sélectionnez la langue, puis cliquez sur Ouvrir. 
3. Faites les traductions de la page, puis cliquez sur **Publier.**
